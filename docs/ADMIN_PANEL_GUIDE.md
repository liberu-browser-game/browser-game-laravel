# Filament Admin Panel - User Guide

## Overview

The admin panel has been enhanced with improved navigation, comprehensive widgets, and powerful content management features to streamline game administration.

## Key Features

### 1. Dashboard Overview

The dashboard provides a comprehensive overview of your game's current state with:

- **Game Statistics Widget**: Real-time stats showing total players, average level, active guilds, and quests in progress
- **Player Level Distribution Chart**: Visual breakdown of player levels across different ranges
- **Content Statistics**: Pie chart showing distribution of items by category
- **Item Type Chart**: Visual representation of item types
- **Recent Players Table**: Quick view of the 10 most recent player registrations
- **Quick Actions Widget**: Fast access to common administrative tasks

### 2. Navigation Structure

Resources are organized into logical groups:

#### Game Management
- **Players** (Sort: 1): Manage player accounts, levels, and progression
- **Items** (Sort: 2): Create and manage in-game items
- **Quests** (Sort: 3): Design and configure quests
- **Player Resources** (Sort: 4): Monitor player resources (gold, wood, etc.)
- **Guilds** (Sort: 5): Manage guilds and their memberships

#### System
- **Game Settings** (Sort: 1): Configure game mechanics and parameters
- **General Settings** (Sort: 3): Site-wide settings and configuration

### 3. Global Search

Access global search with keyboard shortcuts:
- **Mac**: `Cmd + K`
- **Windows/Linux**: `Ctrl + K`

Search across:
- Players (by username, email, level)
- Items (by name, type, rarity)
- Quests (by name, description)
- Guilds (by name, description)

### 4. Player Management

Enhanced player resource with:

#### View Options
- **List View**: Browse all players with filters
- **Detail View**: Complete player information
- **Edit View**: Modify player data

#### Relation Managers
- **Inventory**: View and manage player items
- **Quests**: Track player quest progress
- **Resources**: Monitor player resources (gold, wood, stone, etc.)

#### Filters
- High Level Players (10+)
- Level Ranges: Beginner, Intermediate, Advanced, Expert

#### Bulk Actions
- **Delete**: Remove multiple players
- **Adjust Level**: Increase or decrease levels for multiple players

### 5. Item Management

#### Features
- Type and rarity-based organization
- Color-coded badges for visual clarity
- Duplicate action to create item copies
- Rich descriptions with tooltips

#### Bulk Actions
- **Delete**: Remove multiple items
- **Update Rarity**: Change rarity for multiple items at once

#### Filters
- Filter by Type: Weapon, Armor, Consumable, Material, Quest Item, Misc
- Filter by Rarity: Common, Uncommon, Rare, Epic, Legendary

### 6. Quest Management

#### Features
- Comprehensive reward configuration
- Item and experience rewards
- Duplicate action for creating similar quests
- Description tooltips

#### Form Sections
- **Quest Details**: Name and description
- **Rewards**: Experience and item rewards

#### Filters
- Has Item Reward
- High XP Reward (100+)

### 7. Game Settings

Configure core game parameters:

#### Player Settings
- Enable/disable player registration
- Starting level (default: 1)
- Maximum player level (default: 100)
- Starting gold (default: 100)
- Starting experience (default: 0)

#### Game Mechanics
- Experience multiplier (1-10x)
- Gold multiplier (1-10x)
- Maximum inventory size (default: 100)
- Maximum active quests (default: 10)

#### Features
- Enable/disable PvP
- Enable/disable guild system

### 8. Best Practices

#### Content Management
1. Use the duplicate action to create similar items/quests quickly
2. Use bulk actions to update multiple records efficiently
3. Leverage filters to find specific content
4. Use global search for quick navigation

#### Player Management
1. Use level range filters to segment players
2. Monitor player progression via the dashboard
3. Use relation managers to view player inventory and quests
4. Use bulk level adjustment for game events

#### Settings Management
1. Configure game settings before launch
2. Adjust multipliers for special events
3. Monitor the impact of settings changes via dashboard widgets

## Tips and Tricks

1. **Keyboard Navigation**: Use global search (Cmd+K / Ctrl+K) for faster navigation
2. **Bulk Operations**: Select multiple records to perform batch operations
3. **Filters**: Combine multiple filters for precise data queries
4. **Dashboard Widgets**: Monitor key metrics at a glance
5. **Helper Text**: Hover over form fields to see helpful tooltips
